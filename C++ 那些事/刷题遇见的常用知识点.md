
# 最大公约数

```cpp
int gcd(int a, int b)
{
    while (b != 0)
    {
        int tmp = a;
        a = b;
        b = tmp % b;
    }
    return a;
}
```

或者更简洁

```cpp
int gcd(int a,int b)
{
    return b?gcd(b,a%b):a;
}
```


# 判断一个数是不是质数

```cpp
bool is_prime(int x) {
    if (x <= 1) {
        return false;
    }
    for (int i = 2; i <= x / i; i++) {
        if (x % i == 0) {
            return false;
        }
    }
    return true;
}
```
