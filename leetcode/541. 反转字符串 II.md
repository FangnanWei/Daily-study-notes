```cpp
class Solution {
public:
    string reverseStr(string s, int k) {
        for(int i = 0;i<s.size();i += (2*k))
            {
                int left = i;
                int right = i + k - 1;
                if((s.size()-k)<left)
                    {
                        left = i;
                        right = s.size()-1;
                    }
                while(left<right)
                    {
                        swap(s[left],s[right]);
                        left++;
                        right--;
                    }
            }
        return s;
    }
};
```
s.size()函数返回值的实际类型是string::size_type，该类型是一个无符号整型数。在表达式中混用unsigned int与int可能产生意想不到的结果。
  例如s.size()-3，由于s.size()返回无符号数，故当s.size()<3时，s.size()-3依旧是正数，而非负数。
